# Trang web bán lẻ hàng công nghệ TnTech

* Front-end:
Các bước cài đặt và sử dụng:
- Clone git
- Mở terminal:
+ cd TnTech
+ npm install / npm i
+ npm run dev

 * Back-end: API
 - Mở terminal: 
 + Clone git
 + npm install / npm i
 + node server.js localhost:5000/

 * Database
 + Run query tạo Database thiết lập env trong file .env (đã thiết lập cho PostgreeSQL)
